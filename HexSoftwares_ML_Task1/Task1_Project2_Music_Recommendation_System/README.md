# Music Recommendation System

## Overview

This project recommends songs based on genre similarity using Machine Learning techniques.

## Technologies Used

- Python
- Pandas
- Scikit-Learn

## Algorithm

- CountVectorizer
- Cosine Similarity

## Dataset

songs.csv

## Features

- Song Recommendation
- Genre-Based Similarity
- Content-Based Filtering

## Output

Suggests songs similar to the selected song.
