This repository contains all Machine Learning projects completed during the **HexSoftwares Machine Learning Projects**.

The repository consists of **3 Projects tasks** containing **9 complete Machine Learning projects**.

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Prediction
- Result Visualization

- Positive
- Negative
- Neutral

### Technologies

- Python
- Pandas
- Scikit-Learn
- NLP

### Algorithm

- CountVectorizer
- Multinomial Naive Bayes

### Project 2 — Music Recommendation System

Developed a recommendation engine that suggests songs based on genre similarity using Content-Based Filtering.

### Technologies

- Python
- Pandas
- Scikit-Learn

### Techniques

- CountVectorizer
- Cosine Similarity

# ▶️ How to Run

## 1 Clone Repository

```bash
git clone https://github.com/Armi-123/HexSoftwares_Machine_learning.git
```

---

## 2 Navigate

```bash
cd HexSoftwares_Machine_learning
```

---

## 3 Install Libraries

```bash
pip install -r requirements.txt
```

or

```bash
pip install pandas numpy matplotlib seaborn scikit-learn tensorflow keras nltk
```

---

## 4 Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the required project notebook and run all cells.
