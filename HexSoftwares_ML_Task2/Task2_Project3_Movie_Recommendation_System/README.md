# Movie Recommendation System

## Overview

This project recommends movies based on content similarity using Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Machine Learning Techniques

* TF-IDF Vectorization
* Cosine Similarity
* Content-Based Filtering

## Dataset

movies.csv

Dataset contains:

* Movie Title
* Genre
* Overview
* Popularity
* Release Date
* Vote Average
* Vote Count

## Features

* Movie Recommendation
* Similarity Analysis
* Content-Based Filtering
* Top Rated Movies Visualization

## Output

The system recommends movies similar to a selected movie.

## Conclusion

This project demonstrates how Machine Learning can be used to build personalized movie recommendation systems and improve user experience.
