# Stock Price Prediction using Machine Learning

## Overview

This project predicts future stock prices using historical stock market data and Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Algorithm

* Linear Regression

## Features

* Data Loading
* Data Visualization
* Model Training
* Future Stock Price Prediction

## Dataset

stock_data.csv

## Output

Predicts future stock prices based on historical trends.

## Conclusion

The project demonstrates the use of Linear Regression for forecasting stock prices and understanding market trends.
