
# News Article Summarization using NLP

## Overview

This project automatically summarizes long news articles into concise summaries using Natural Language Processing (NLP).

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-Learn

## Features

* Automatic News Summarization
* Text Preprocessing
* Frequency-Based Sentence Ranking
* Summary Generation

## Dataset

Articles.csv

Dataset Columns:

* Article
* Date
* Heading
* NewsType

## Workflow

1. Load News Articles
2. Clean Text
3. Remove Stopwords
4. Calculate Word Frequencies
5. Score Sentences
6. Generate Summary

## Output

The system generates concise summaries while retaining the most important information from the original article.

## Conclusion

This project demonstrates the use of NLP techniques to automate text summarization, making large volumes of news content easier to consume and understand.
