# Personality Prediction System Through CV Analysis

## Overview

This project predicts personality traits from resume/CV text using Natural Language Processing (NLP) and Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLTK

## Machine Learning Techniques

* TF-IDF Vectorization
* Multinomial Naive Bayes

## Features

* Resume Analysis
* Personality Prediction
* NLP-Based Text Processing
* Automated Candidate Assessment

## Dataset

resumes.csv

## Personality Traits

* Openness
* Conscientiousness
* Extraversion
* Agreeableness
* Emotional Stability

## Output

Predicts the most likely personality trait based on resume content.

## Conclusion

The system demonstrates how NLP and Machine Learning can be used to automate personality assessment from professional resumes, helping organizations improve recruitment efficiency.
