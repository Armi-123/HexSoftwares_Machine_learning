# Malaria Detection using Machine Learning

## Overview

This project predicts malaria infection using Machine Learning algorithms trained on blood sample features.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn


## Algorithm

- Random Forest Classifier

## Dataset

malaria.csv

Dataset Size:
- 27,000+ Records

Target Classes:
- Parasitized
- Uninfected

## Features

- Data Analysis
- Classification
- Confusion Matrix
- Disease Prediction
- Accuracy Evaluation

## Output

Predicts whether a blood sample is infected with malaria parasites.

## Conclusion

The project demonstrates the application of Machine Learning in healthcare for automated malaria detection and early diagnosis support.
